# Flight Booking System in C++

A simple airport ticket booking system developed in C++ using Object-Oriented Programming concepts.

## Features
- View available flights
- Book flight tickets
- Store ticket records in file
- Display booked ticket details
- Menu-driven system

## Concepts Used
- Classes & Objects
- Encapsulation
- Arrays
- Functions
- File Handling
- Conditional Statements
- Loops

## Technologies
- C++
- OOP Concepts
